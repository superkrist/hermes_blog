## 欢迎来到我的博客！

经过一番折腾，终于用 **Hugo + GitHub Pages** 搭建好了这个博客站点。

### 技术栈

- **静态站点生成器**: [Hugo](https://gohugo.io/) (Extended v0.128.0)
- **主题**: [PaperMod](https://github.com/adityatelange/hugo-PaperMod)
- **托管**: GitHub Pages
- **CI/CD**: GitHub Actions — 合并到 `main` 分支自动部署

### 博客内容规划

| 方向 | 内容 |
|------|------|
| 🖥️ 技术 | 后端开发、DevOps、工具链 |
| 📊 量化交易 | 回测框架、因子策略、A 股分析 |
| 🤖 机器学习 | XGBoost、LightGBM、CatBoost 实战 |
| 📝 笔记 | 各技术领域的总结与踩坑记录 |

### 部署流程

```
写文章 → content/posts/ → 提交 PR → merge 到 main → 自动部署 🚀
```

后续会陆续更新内容，敬请期待！
